# Focus With Music Backend


## Initializing
1. clone repository
2. `$ cd <repository path>`
3. `$ python3 -m venv venv`
4. `$ source venv/bin/activate`
5. `$ pip install -r requirements.txt`
6. flaskenv 
7. env 
8. config.py
9. `$ flask db migrate -m "init migrate"`
10. `$ flask db upgrade`
11. `$ flask run`
